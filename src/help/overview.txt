shard — encrypted thought store

Usage:
  shard <command> [options]

Commands:
  init         Bootstrap a new workspace (run this first)
  daemon       Start the daemon (manages all shards in-process)
  new          Create a new shard (interactive wizard)
  connect      Open a session to the daemon or a standalone shard
  mcp          Start the MCP server (JSON-RPC over stdio)
  vault        Export all shards as Obsidian markdown vault
  help         Show help for a command

Standalone shard (for remote/federated use):
  shard --name <n> --key <hex> [--data <path>] [--timeout <s>]

Global flags:
  --help, -h     Show help
  --ai-help      Print structured markdown reference for AI agents

Use "shard help <command>" for detailed help on any command.
Use "shard help shard" for the full operation reference with examples.

AI AGENTS: Run "shard --ai-help" for a structured markdown reference designed
for machine consumption. It contains the complete operation reference, message
format specification, workflow instructions, and examples — everything needed
to operate shards programmatically. Start there instead of parsing this text.
