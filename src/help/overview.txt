shard — encrypted thought store

Usage:
  shard <command> [options]

Commands:
  init         Bootstrap a new workspace (run this first)
  install      Workspace setup + AI agent setup reference (--ai for agents)
  daemon       Start the daemon (manages all shards in-process)
  new          Create a new shard (interactive wizard)
  connect      Open a session to the daemon or a standalone shard
  mcp          Start the MCP server (JSON-RPC over stdio or --http with sse)
  dump         Export shards as markdown files

Standalone shard (for remote/federated use):
  shard --name <n> --key <hex> [--data <path>] [--timeout <s>]

Flags:
  --help, -h     Show help for any command
  --ai           Print structured markdown reference for AI agents

Examples:
  shard daemon --help      Human help for the daemon command
  shard mcp --ai           AI agent reference for MCP tools
  shard --ai               Full AI agent reference (start here)
