shard install — workspace setup + AI agent setup reference

Usage:
  shard install [--help] [--ai]

What it does:
  Initializes .shards/ workspace (same as shard init), then prints
  a pointer to the AI agent setup reference.

Flags:
  --help       Show this help
  --ai         Print full AI agent setup guide (tool config, MCP setup,
               step-by-step instructions for agents)

AI AGENTS: Run "shard install --ai" for step-by-step setup instructions.
The guide covers: PATH setup, workspace init, MCP config per tool, and verification.
